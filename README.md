# 🏢 건축물 온실가스 계산기 v2.7.1

## 📋 소개

건축물에서 발생하는 온실가스 배출량을 정확하게 계산할 수 있는 웹 기반 계산기입니다.  
**Scope 1** (직접 배출), **Scope 2** (간접 배출), **Scope 3** (기타 간접 배출)를 모두 지원하며,  
대한민국 온실가스 배출권거래제 지침에 따른 국가 고유 배출계수를 적용합니다.

## ✨ 주요 기능

### 🔥 Scope 1 - 직접 배출 (연료 연소)
- 가스류: 천연가스(LNG), 도시가스(LNG/LPG)
- 석유류: 휘발유, 등유, 경유, B-A/B/C유
- 석탄류: 국내 무연탄, 수입 무연탄, 유연탄
- 바이오연료: 목재펠릿, 우드칩, 바이오가스, 바이오디젤 등
- Tier 1/2/3 선택 가능

### ⚡ Scope 2 - 간접 배출 (전기/열)
- 전기(소비단/발전단)
- 열(열전용/열병합/열평균)
- 지역난방 (8개 지역, 3기/4기 계획기간)
- **전력 배출계수 연도별 선택** (ETS 3기/4기, 2017~2022 실측)

### 🚗 Scope 3 - 기타 간접 배출
- **출장**: 항공, KTX, 버스, 승용차 등
- **통근**: 편도 거리 × 일수 × 인원 (왕복 자동 계산)
- **폐기물**: 일반/음식물 (매립/소각/퇴비화)
- **용수**: 상수, 중수, 하수
- 배출계수 기준: 추정치 / DEFRA 2024 / 연료 기반 선택

### 📊 분석 기능
- **원단위 분석**: 면적당 (kgCO2eq/m²·년), 인당 (tCO2eq/인·년)
- **벤치마크 비교**: 건물 유형별 평균 대비 수준
- **녹색건축 인증 예측**: G-SEED 점수, ZEB 등급
- **차트 시각화**: Scope 비율 (파이), 항목별 비교 (막대), 시계열 트렌드 (라인)

### 💾 데이터 관리
- **자동 저장**: localStorage에 자동 저장
- **JSON 내보내기/가져오기**: 백업 및 복원
- **Excel 내보내기**: CSV 형식
- **PDF 리포트**: 종합 보고서 생성

### 📚 내장 가이드
- **완벽 가이드**: 온실가스 계산 이론 및 개념
- **사용자 매뉴얼**: 웹앱 사용법

## 🚀 시작하기

### 방법 1: 직접 열기
`ghg_calculator_v2/index.html` 파일을 브라우저로 열기

### 방법 2: 로컬 서버
```bash
cd ghg_calculator_v2
python -m http.server 8080
```
브라우저에서 `http://localhost:8080` 접속

## 🏠 주거용 vs 상업용/공공 분석

### ✅ 결론: 둘 다 지원됨!

Excel 계산기는 **주거용**과 **상업용/공공** 모두 지원합니다.

### 적용 위치
- **Main!H3 셀**: "주거용" 또는 "상업용/공공" 선택 (드롭다운)

### 부문별 차이점

#### 석탄류 CH4 배출계수 (kgCH4/TJ)
| 연료 | 상업용/공공 | 주거용 |
|------|------------|--------|
| 국내 무연탄 | 10 | 300 |
| 수입 무연탄 | 10 | 300 |
| 원료용 수입 무연탄 | 10 | 300 |
| 원료용 유연탄 (점결탄) | 10 | 300 |
| 연료용 유연탄 | 10 | 300 |
| 아역청탄 | 10 | 300 |
| 갈탄 | 10 | 300 |
| 기타 석탄류 | 10 | 300 |

### 왜 차이가 나는가?
IPCC 2006 가이드라인 제2권 (에너지)에 따르면:

> **주거용 연소**는 일반적으로 소규모, 저효율 연소 설비를 사용하므로 불완전 연소로 인한 CH4 배출이 높음 (300 kgCH4/TJ)
> 
> **상업용/공공**은 상대적으로 효율적인 연소 설비를 사용하므로 CH4 배출이 낮음 (10 kgCH4/TJ)

### 수식 확인
```excel
=IF(Main!$H$3="상업용/공공", 10, IF(Main!$H$3="주거용", 300))
```

이 수식이 석탄류 연료의 K열(CH4 배출계수)에 적용됨.

## 📌 웹 앱 구현 시 추가해야 할 것

### 1. 주거용/상업용 선택 기능
현재 프로토타입에는 이 선택 옵션이 **누락**되어 있음!

```html
<select id="buildingType">
    <option value="주거용">주거용</option>
    <option value="상업용/공공">상업용/공공</option>
</select>
```

### 2. 석탄류 CH4 배출계수 조건부 적용
```javascript
function getCH4EmissionFactor(fuel, buildingType) {
    if (fuel.category === "석탄류") {
        return buildingType === "상업용/공공" ? 10 : 300;
    }
    return fuel.ch4EmissionFactor;
}
```

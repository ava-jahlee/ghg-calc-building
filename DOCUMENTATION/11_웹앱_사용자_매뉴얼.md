## 3. 기본 설정 방법

### 📝 설정 항목

#### 1) 분야/부문
- 자동 설정: "1A. 에너지" / "1A4. 기타"
- 건축물 배출은 이 분류에 해당

#### 2) 세부 (건물 용도)
| 옵션 | 설명 | 영향 |
|------|------|------|
| **주거용** | 아파트, 주택 등 | 석탄류 CH4 = 300 |
| **상업용/공공** | 사무실, 공공건물 등 | 석탄류 CH4 = 10 |

> ⚠️ **중요**: 석탄류 연료 사용 시 이 설정에 따라 CH4 배출량이 30배 차이남!

#### 3) 건물 정보 (원단위 분석용)
| 항목 | 설명 | 용도 |
|------|------|------|
| **건물 유형** | 사무실, 학교, 병원 등 9종 | 벤치마크 비교 기준 |
| **연면적 (m²)** | 건물 총 면적 | 면적당 배출량 계산 |
| **재실인원 (명)** | 상주 인원 수 | 인당 배출량 계산 |

> 💡 **도움말 버튼 (📊)**: 건물 유형별 에너지 소비 참고값 확인 가능

#### 4) 배출계수/열량계수 기준
| 옵션 | 설명 |
|------|------|
| **국가고유(17년)** | 2017년 환경부 고시 값 |
| **국가고유(22년)** | 2022년 환경부 고시 값 (최신) |

#### 5) GWP 기준
| 옵션 | CH4 | N2O | 용도 |
|------|-----|-----|------|
| **국가 인벤토리 (SAR)** | 21 | 310 | 국내 공식 보고 |
| **IPCC AR4 (2007)** | 25 | 298 | 국제 기준 |
| **IPCC AR5 (2014)** | 28 | 265 | 국제 기준 |
| **IPCC AR6 (2021)** | 27.9 | 273 | 최신 기준 |

#### 6) Tier 최소 기준
- 시설규모에 따라 자동 표시
- 예: A등급 → 열량: T2, 배출: T1, 산화: T1

## 4. Scope 1 계산하기

### 🔥 연료 연소 배출량 계산

#### Step 1: 연료 선택
드롭다운에서 연료 선택:
- **가스류**: 천연가스(LNG), 도시가스(LNG/LPG)
- **석유류**: LPG, 휘발유, 등유, 경유, B-A/B/C유
- **석탄류**: 국내 무연탄, 수입 무연탄, 유연탄
- **바이오연료**: 목재펠릿, 우드칩, 폐목재, 바이오가스, 바이오디젤, 바이오에탄올

#### Step 2: 연료 상태 확인
- **Default**: 연료별 기본 상태 자동 적용
- 수동 선택 가능: 고체 / 액체 / 기체

> 💡 상태에 따라 산화계수가 달라집니다!
> - 고체: 0.98
> - 액체: 0.99
> - 기체: 0.995

#### Step 3: 사용량 입력
- 숫자 입력
- 단위는 연료에 따라 자동 설정 (ton, kL, 천m³)

#### Step 4: Tier 선택
| Tier | 열량 | 배출계수 | 산화계수 |
|------|------|----------|----------|
| **T1 (IPCC)** | 국제 기본값 | 국제 기본값 | 1.0 |
| **T2 (국가)** | 국가 고유값 | 국가 고유값 | 연료상태별 |
| **T3 (직접입력)** | 직접 입력 | 직접 입력 | 직접 입력 |

#### Step 5: 결과 확인
```
┌────────┬────────┬────────┬────────┐
│  CO2   │  CH4   │  N2O   │  합계  │
│ 2.7713 │4.94e-5 │4.94e-6 │ 2.7739 │
│ tCO2eq │  tCH4  │  tN2O  │ tCO2eq │
└────────┴────────┴────────┴────────┘
```

#### Step 6: 목록에 추가
**[+ 추가]** 버튼 클릭 → 우측 총 배출량 목록에 추가됨

#### 탭 구성
| 탭 | 항목 | 단위 |
|-----|------|------|
| **✈️ 출장** | 업무 출장 이동 | 인·km |
| **🚌 통근** | 직원 출퇴근 | km, 일, 명 |
| **🗑️ 폐기물** | 건물 발생 폐기물 | ton |
| **💧 용수** | 상수/중수/하수 | m³ |

### ✈️ 출장 계산

#### 교통 배출계수 기준 선택
| 옵션 | 특징 | 권장 상황 |
|------|------|----------|
| **⚠️ 추정치** | 대략적 규모 파악 | 공식 자료 없을 때 |
| **DEFRA 2024** | 영국 정부 공식 | 국제 기준 필요 시 |
| **연료 기반** | 연비 입력 방식 | 정확도 필요 시 |

#### 교통수단 종류
- **항공**: 국내선, 국제-단거리, 국제-장거리
- **철도**: KTX/SRT, 일반열차
- **버스**: 시외/고속버스
- **승용차**: 휘발유, 경유, LPG, 하이브리드, 전기

### 🚌 통근 계산
- **편도 거리**: 집-회사 편도 거리 (km)
- **출근 일수**: 연간 출근 일수
- **인원**: 해당 교통수단 이용 인원

> 💡 통근은 **왕복 자동 계산**됩니다 (편도 × 2)

## 8. 원단위 분석 및 벤치마크

### 📐 원단위 분석
| 지표 | 설명 | 단위 |
|------|------|------|
| **📐 면적당** | 총 배출량 ÷ 연면적 | kgCO2eq/m²·년 |
| **👤 인당** | 총 배출량 ÷ 재실인원 | tCO2eq/인·년 |

### 🎯 벤치마크 비교
선택한 건물 유형의 평균 대비 배출 수준을 표시:
- **우수**: 평균보다 현저히 낮음
- **평균**: 일반적 수준
- **개선필요**: 평균보다 높음

> 💡 면적과 재실인원을 입력해야 원단위 분석이 표시됩니다.

## 9. 녹색건축 인증 예측

### 🌿 G-SEED (녹색건축인증)
온실가스 배출량 기준으로 점수 예측:
| 등급 | 점수 |
|------|------|
| **최우수** | 80점 이상 |
| **우수** | 70~79점 |
| **우량** | 60~69점 |
| **일반** | 50~59점 |

### ⚡ ZEB (제로에너지빌딩)
신재생에너지 생산량을 입력하면 에너지자립률과 ZEB 등급 예측:
| 등급 | 에너지자립률 |
|------|-------------|
| **1등급** | 100% 이상 |
| **2등급** | 80~100% |
| **3등급** | 60~80% |
| **4등급** | 40~60% |
| **5등급** | 20~40% |

> 💡 ZEB 예측을 위해 "신재생에너지 생산량(kWh/년)"을 입력하세요.

## 11. 데이터 저장/불러오기

### 💾 자동 저장
- 모든 변경사항은 **자동으로 localStorage에 저장**됩니다
- 브라우저를 닫았다 열어도 데이터 유지

### 💾 파일로 저장
1. **[💾 저장]** 버튼 클릭
2. JSON 파일로 다운로드
3. 다른 컴퓨터나 백업용으로 활용

### 📂 파일 불러오기

## 14. 주의사항 및 문제해결

### ⚠️ 주의사항

#### 1. 석탄류 사용 시
- 반드시 **세부(주거용/상업용)** 설정 확인
- CH4 배출량이 30배 차이남

#### 2. Tier 선택 시
- 시설규모별 최소 기준 확인
- 배출계수 Tier 변경 시 산화계수도 자동 변경

#### 3. 지역난방 사용 시
- 정확한 지역 선택 필수
- 계획기간(3기/4기) 확인

#### 4. Scope 3 교통 배출계수
- "추정치"는 공식 출처 미확인 - 대략적 규모 파악용
- 정확한 계산이 필요하면 "DEFRA 2024" 또는 "연료 기반" 사용

#### 5. 전력 배출계수
- **배출권거래제 보고**: ETS 계획기간별 값 사용 (3기/4기)
- **실제 추적**: 연도별 실측값 사용

### 🆘 문제 해결

#### Q: 계산 결과가 0으로 나옵니다
**A**: 사용량이 0으로 입력되어 있는지 확인하세요.

#### Q: 추가 버튼이 작동하지 않습니다
**A**: 모든 필수 항목이 선택되었는지 확인하세요.

#### Q: 원단위 분석이 표시되지 않습니다
**A**: 건물 정보(연면적, 재실인원)를 입력하고, 항목을 1개 이상 추가하세요.

#### Q: 입력값 경고가 표시됩니다
**A**: 입력한 사용량이 건물 유형 기준 참고값에서 벗어났다는 의미입니다. 
입력값을 다시 확인하거나, 실제 값이 맞다면 무시해도 됩니다.

#### Q: 데이터가 사라졌습니다
**A**: 
1. 브라우저의 데이터 삭제(캐시 삭제)를 했다면 복구 불가
2. JSON 파일로 백업해두면 복원 가능

#### Q: PDF가 생성되지 않습니다
**A**: 인터넷 연결 상태를 확인하세요 (jsPDF 라이브러리 로딩 필요).

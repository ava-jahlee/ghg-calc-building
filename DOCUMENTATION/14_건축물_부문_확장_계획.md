```

│   └── 1A4. 기타 부문 ⭐

│       │   ├── 병원/의료시설
│       │   ├── 호텔/숙박시설
│       │   ├── 쇼핑몰/상가
│       │   ├── 체육시설
│       │   └── 공공기관
│       └── 1A4b. 주거 ✅ 구현됨

│           ├── 단독주택
```

## 🎯 확장 계획 (우선순위순)

### Phase 1: 건축물 특화 기능 (v2.5)

#### 1.1 원단위 배출량 계산
> 건물 규모 대비 효율 비교

| 원단위 | 계산식 | 용도 |
|--------|--------|------|
| **면적당 배출량** | 총배출량 ÷ 연면적(m²) | 건물 효율 비교 |
| **재실자당 배출량** | 총배출량 ÷ 재실인원 | 인원 대비 효율 |
| **층당 배출량** | 총배출량 ÷ 층수 | 수직 분포 분석 |

**구현 내용:**
```javascript
// 기본 설정에 건물 정보 입력
- 연면적 (m²)
- 재실인원 (명)
- 층수
- 준공연도

// 결과에 원단위 표시
- XX kgCO2eq/m²·년
- XX tCO2eq/인·년
```

#### 1.2 건축물 유형별 벤치마크
> 동종 건물 대비 성능 비교

| 건물 유형 | 벤치마크 (kgCO2eq/m²·년) | 출처 |
|-----------|--------------------------|------|
| 사무실 | 50-80 | 환경부 |
| 학교 | 30-50 | 환경부 |
| 병원 | 100-150 | 환경부 |
| 호텔 | 80-120 | 환경부 |
| 쇼핑몰 | 60-100 | 환경부 |
| 아파트 | 20-40 | 환경부 |

**구현 내용:**
- 건물 유형 선택 → 벤치마크 자동 표시
- 내 건물 vs 평균 비교 게이지
- 상위/하위 % 표시

#### 1.3 에너지 효율 등급 연계
> 건축물 에너지효율등급 인증제

| 등급 | 1차에너지소요량 (kWh/m²·년) | CO2 기준 |
|------|------------------------------|----------|
| 1+++ | < 60 | 최고 효율 |
| 1++ | 60-90 | 매우 우수 |
| 1+ | 90-120 | 우수 |
| 1 | 120-150 | 양호 |
| 2 | 150-190 | 보통 |
| 3-7 | > 190 | 개선 필요 |

### Phase 2: 녹색건축인증 연계 (v2.6)

#### 2.1 G-SEED (녹색건축인증) 항목 연계

| 평가 항목 | 배점 | 계산기 연계 |
|-----------|------|-------------|
| 에너지 소비량 | 12점 | ✅ 가능 |
| 온실가스 배출량 | 6점 | ✅ 가능 |
| 생활용수 절감 | 4점 | ✅ 가능 (용수) |
| 폐기물 처리 | 2점 | ✅ 가능 (폐기물) |
| 교통 (녹색교통) | 2점 | ✅ 가능 (통근) |

**구현 내용:**
- G-SEED 점수 예측
- 개선 포인트 제안
- 인증 등급 예상 (최우수/우수/우량/일반)

#### 2.2 ZEB (제로에너지빌딩) 등급 시뮬레이션

| ZEB 등급 | 에너지자립률 | 1차에너지소요량 |
|----------|-------------|-----------------|
| 1등급 | 100% 이상 | 0 이하 |
| 2등급 | 80-100% | 60 이하 |
| 3등급 | 60-80% | 90 이하 |
| 4등급 | 40-60% | 120 이하 |
| 5등급 | 20-40% | 140 이하 |

**구현 내용:**
- 신재생에너지 생산량 입력 (태양광 등)
- 에너지자립률 자동 계산
- ZEB 등급 예측
- 등급 향상을 위한 필요 에너지량 제시

|------|-------------------|-------------|

| 사무기기 | 10-15% | 고효율 기기 |

| 기타 | 5-10% | - |

**구현 내용:**
- 설비별 에너지 사용량 입력
- 설비별 배출량 분석
- 우선 개선 대상 제시

#### 4.2 계절별/시간대별 분석

- 월별 에너지 사용 패턴
- 냉방 vs 난방 비율
- 피크 시간대 분석
- 기저 부하 분석

### Phase 5: 탄소중립 시뮬레이션 (v3.0)

#### 5.1 감축 시나리오 분석

```
현재 배출량: 1,000 tCO2eq/년

시나리오 1: LED 교체
- 예상 감축: 50 tCO2eq/년 (5%)
- 투자비용: 2,000만원
- 회수기간: 4년

시나리오 2: 태양광 설치 (100kW)
- 예상 감축: 80 tCO2eq/년 (8%)
- 투자비용: 1.5억원
- 회수기간: 8년

시나리오 3: 냉난방 고효율화
- 예상 감축: 100 tCO2eq/년 (10%)
- 투자비용: 5억원
- 회수기간: 10년
```

#### 5.2 탄소중립 로드맵

- 2030년 목표: 40% 감축
- 2050년 목표: 탄소중립
- 연도별 감축 경로 시각화
- 필요 투자 규모 산정

- [ ] 검증 테스트

## 📅 Phase 3: 데스크톱 앱 (업무용)

### 3.1 기술 스택 선정
**옵션 1: Electron**
- 장점: 웹 코드 재사용 가능
- 단점: 용량이 큼 (100MB+)

**옵션 2: Python + PyQt/Tkinter**
- 장점: 가벼움, Python 생태계 활용
- 단점: 웹 코드 재작성 필요

**옵션 3: Tauri**
- 장점: 가벼움, Rust 기반
- 단점: 학습 곡선

**권장: Electron** (웹 코드 최대 재사용)

### 3.2 추가 기능
- [ ] 대량 데이터 일괄 계산
- [ ] Excel 파일 직접 읽기/쓰기
- [ ] 데이터베이스 연동
- [ ] 자동화 스크립트 지원
- [ ] 분석 리포트 자동 생성

### 예상 소요 시간: 5-7일

## ⚠️ 중요 체크포인트

### 계산 정확성 검증
1. Excel 파일에서 테스트 케이스 생성
2. 웹 앱에서 동일한 입력값으로 계산
3. 결과 비교 (허용 오차: ±0.0001)
4. 불일치 시 원인 분석 및 수정

### 데이터 무결성
1. Excel 데이터와 JavaScript 데이터 1:1 대응
2. 연료명 정확히 일치 (띄어쓰기, 괄호 포함)
3. 숫자 데이터 정확성 검증

### 문서화 품질
1. 모든 수식에 주석
2. 계산 로직 흐름도
3. 다른 개발자/LLM이 이해할 수 있는 수준

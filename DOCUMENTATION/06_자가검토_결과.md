## ⚠️ 수정 필요 항목

### 1. 열 구조 설명 보완 필요

**문제:** 데이터테이블 문서의 열 번호가 **VLOOKUP 범위 기준**과 **실제 시트 열**이 혼용됨

**현재 문서 (03_데이터테이블.md):**
```
| N | 열량계수 T1 - 17년 (MJ/kg) |
| O | 열량계수 T1 - 22년 (MJ/kg) |
```

**실제 구조:**
- VLOOKUP 범위: `'_Law&GL22'!$B$35:$U$100` (B열 = 1열)
- VLOOKUP 14열 = O열 (시트 15열) = 순발열량 (열량계수 T1)
- VLOOKUP 16열 = Q열 (시트 17열) = 순발열량 (열량계수 T2)

**수정 방향:**
열 구조를 **실제 시트 열(A, B, C...)** 기준으로 명확하게 재정리

### 2. 열량계수 열 번호 수정

**현재 (02_Main시트_분석.md):**
> T1: IPCC 기본값 (열량계수 14열, 배출계수 3/4/5열)
> T2: 국가고유값 (열량계수 16열, 배출계수 9/10/11열)

**수정 필요:**
이 열 번호는 **VLOOKUP 범위 내 상대 열 번호**임을 명시
- VLOOKUP 14열 = 시트 O열
- VLOOKUP 16열 = 시트 Q열
- VLOOKUP 3/4/5열 = 시트 D/E/F열
- VLOOKUP 9/10/11열 = 시트 J/K/L열

### 3. T1/T2 열량계수 혼동

**문서에서:**
> 열량계수 T1 - 17년, T1 - 22년

**실제:**
- O열: 순발열량 (IPCC 기본값) → 이것이 T1
- P열: 총발열량
- Q열: 순발열량 (국가고유값) → 이것이 T2

17년/22년 구분은 **배출계수 기준년도**이며, 열량계수 T1은 연도와 무관한 IPCC 기본값

## 📊 테스트 케이스 계산 재검증

### 테스트 케이스 1: 기본값 (천연가스 1ton, T2)

```
입력:
  연료: 천연가스(LNG)
  사용량: 1 ton
  열량계수 Tier: T2
  배출계수 Tier: T2
  산화계수 Tier: T2

매개변수:
  열량계수: 49.4 MJ/kg
  CO2 배출계수: 56,144 kgCO2/TJ
  CH4 배출계수: 5 kgCH4/TJ
  N2O 배출계수: 0.1 kgN2O/TJ
  산화계수: 0.995 (기체)

계산:
  CO2 = 0.000001 × 49.4 × 56144 × 0.995 × 1 = 2.759646 tCO2
  CH4 = 0.000001 × 49.4 × 5 × 1 = 0.000247 tCH4 (산화계수 미적용!)
  N2O = 0.000001 × 49.4 × 0.1 × 1 = 0.00000494 tN2O (산화계수 미적용!)

tCO2eq:
  CO2_eq = 2.759646 × 1 = 2.759646
  CH4_eq = 0.000247 × 21 = 0.005187
  N2O_eq = 0.00000494 × 310 = 0.001531

총합: 2.759646 + 0.005187 + 0.001531 = 2.766364 tCO2eq
```

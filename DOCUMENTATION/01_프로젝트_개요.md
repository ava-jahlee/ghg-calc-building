# 온실가스 계산기 (GHGCalc) 프로젝트 문서

## 📋 프로젝트 개요

### 목적
- 건축물의 온실가스(GHG) 배출량을 IPCC 2006 가이드라인에 따라 정확하게 계산
- Excel 기반 계산기를 웹/데스크톱 애플리케이션으로 변환

### 버전
- 원본: `GHGCalc_V0m_lja.xlsm` (Excel 매크로 사용 통합 문서)
- 목표: 웹 앱 (배포용) + 데스크톱 프로그램 (업무용)

### 적용 기준
- IPCC 2006 국가 온실가스 인벤토리 가이드라인
- 국가 고유 발열량 및 배출계수 (17년/22년)
- 온실가스 배출권거래제의 배출량 보고 및 인증에 관한 지침

## 📁 Excel 파일 구조

### 시트 구성
| 시트명 | 크기 | 역할 |
|--------|------|------|
| Main | 43행 × 17열 | 사용자 인터페이스 + 계산 로직 |
| _Law&GL22 | 125행 × 23열 | 법/가이드라인 데이터 (배출계수, 열량계수) |
| _Supplier | 21행 × 9열 | 한국지역난방공사 열(스팀) 배출계수 |

### 수식 통계
| 시트명 | 수식 개수 |
|--------|-----------|
| Main | 44개 |
| _Law&GL22 | 260개 |
| _Supplier | 48개 |

---

## 🔥 계산 범위

### Scope 1: 직접 배출량 (연료 연소)
- 고정연소: 고체/액체/기체 연료
- 대상 온실가스: CO2, CH4, N2O

### Scope 2: 간접 배출량 (외부 에너지 구매)
- 전기: 발전단/소비단
- 열(스팀): 열전용/열병합/열평균/지역난방
